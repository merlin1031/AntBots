# Spezifikationen
+ Dimensionen max. 10x10x10 cm
+ Gewicht: max. 150g
+ Antrieb: 2S (7,4V)
+ Motor: Geriebemotor 6V, Übersetzung 1:200
+ Räder: Pololu 40x7mm Gummi

# CAD
+ Konstruktion mit SolidWorks 2017/18
+ Slicing und 3D-Druck: Repetier Host und Creality Ender 3
